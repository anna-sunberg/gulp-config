# Usage:

    gulp --cwd <project_name>

# Other commands:
Lint:

    gulp lint

Coffee lint:
    
    gulp coffeelint

CSS lint:

    gulp csslint    

Watch without server:    

    gulp watch

Just the server:

    gulp server

Dev build:

    gulp build-dev

CSS only build:

    gulp css
